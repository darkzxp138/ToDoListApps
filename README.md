# Gestor de Tareas

## Descripción

Una aplicación de consola en C# para gestionar una lista de tareas. Permite agregar, listar, marcar como completadas y eliminar tareas.

## Requisitos

- .NET Framework 4.8
- Visual Studio 2022

## Instrucciones para Compilar y Ejecutar

1. Clona el repositorio:
    ```bash
    git clone <url-del-repositorio>
    ```

2. Abre el proyecto en Visual Studio 2022.

3. Compila el proyecto usando la opción "Build" en el menú.

4. Ejecuta la aplicación usando la opción "Start" en el menú o presionando `F5`.

## Uso

- **Agregar Tarea**: Selecciona la opción 1 y sigue las instrucciones para ingresar la descripción y opcionalmente una fecha límite.
- **Listar Tareas**: Selecciona la opción 2 para ver todas las tareas registradas.
- **Marcar Tarea como Completada**: Selecciona la opción 3 y proporciona el número de la tarea que deseas marcar como completada.
- **Eliminar Tarea**: Selecciona la opción 4 y proporciona el número de la tarea que deseas eliminar.
- **Salir**: Selecciona la opción 5 para salir de la aplicación.
